# BowlerGuard (IPD Prototype) — Model-first Implementation

This folder is a minimal, undergraduate-friendly implementation skeleton aligned with your thesis:
- Data → Simulation/Features → Injury Risk Model (+Fatigue score) → Minimal Flask API → Minimal UI

## Recommended environment (best for undergrad research)
Use **Anaconda** with Jupyter Notebooks.
- It keeps everything reproducible and easy to demonstrate in your viva.

## Quick start
1) Create env:
   conda create -n bowlerguard python=3.11 -y
   conda activate bowlerguard

2) Install deps:
   pip install -r requirements.txt

3) Launch Jupyter:
   jupyter notebook

4) Run notebooks in order:
   notebooks/01_data_audit.ipynb
